# Telecom Network Operations Analytics

This repository contains tools and scripts for telecom network operations analytics, data generation, and outage prediction.

## Folder Structure

The project is structured as follows:

```
├── database/
│   └── schema.sql             # SQL schema for the database
├── data_generation/
│   ├── generate_sites.py      # Script to generate site data
│   ├── generate_logs.py       # Script to generate network log data
│   └── generate_incidents.py  # Script to generate network incidents data
├── analytics/
│   ├── calculate_kpis.py      # Script to calculate network KPIs
│   └── outage_prediction.py   # Script to predict network outages
├── exports/                   # Directory for exported data files
├── dashboard/                 # Directory for dashboard resources and assets
├── screenshots/               # Directory for screenshots of dashboards or analytics
└── README.md                  # Project documentation
```
